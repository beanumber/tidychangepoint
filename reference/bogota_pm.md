# Particulate matter in Bogotá, Colombia

Particulate matter of less than 2.5 microns of diameter in Bogotá,
Colombia.

## Usage

``` r
bogota_pm
```

## Format

An object of class `xts` (inherits from `zoo`) with 1096 rows and 1
columns.

## Details

Daily readings from 2018-2020 are included.

## Examples

``` r
class(bogota_pm)
#> [1] "xts" "zoo"
```
