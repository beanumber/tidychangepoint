# Italian University graduates by disciplinary groups from 1926-2013

Italian University graduates by disciplinary groups during the years
1926-2013.

## Usage

``` r
italy_grads
```

An object of class `tbl_ts` (inherits from `tbl_df`, `tbl`,
`data.frame`) with 88 rows and 17 columns.
